# 📊 Antigravity 黄金ETF监控系统 V2.0

华安黄金ETF (518880.SH) 量化建仓监控与决策引擎

---

## 🚀 快速开始

### 第一次使用（安装依赖）

```bash
cd gold-etf-monitor
pip install -r requirements.txt
```

### 启动程序

直接双击 `gold_monitor.py`，或在终端运行：

```bash
python gold_monitor.py
```

**首次启动时**，程序会自动检测缺少的依赖库并提示安装。

---

## 🖥️ 使用说明

启动后界面显示两个按钮：

| 按钮 | 功能 |
|------|------|
| 🌅 **盘前分析** | 全面采集宏观数据，计算GFI，生成交易建议（每日开盘前使用） |
| 📈 **盘中监控** | 检查黄金24h是否出现>3%回调，评估是否触发加仓（盘中随时使用） |

分析完成后，报告会自动保存到 `reports/` 文件夹。

---

## 📁 报告文件命名规则

```
reports/
├── 20260311_0850_盘前_GFI35.txt    # 开盘前分析报告
├── 20260311_1330_盘中_GFI35.txt    # 盘中监控报告
└── ...
```

命名格式：`YYYYMMDD_HHMM_[盘前/盘中]_GFI{分值}.txt`

---

## 📊 GFI 指数说明

| GFI区间 | 含义 | 操作建议 |
|---------|------|---------|
| ≥ 70 | 极度看多 | 强烈买入（组合1.5%） |
| 50–70 | 偏多 | 逐步建仓（组合0.5%） |
| 30–50 | 中性 | 持有/观望 |
| < 30 | 偏空 | 暂停买入 |
| < 20 + RC>20% | 极度看空 | 减仓提示 |
